# Examples

This directory contains lightweight code examples and notebooks.

## Files

- `demo1_run.py`
  - Minimal Python entry point for loading a config and executing a run.
- `Satellite_to_Chip_Digital_Twin.ipynb`
  - Notebook-style exploration of the satellite-to-chip lane.

## When to use `examples/` vs other folders

- Use `examples/` for small runnable code or notebooks.
- Use `configs/` for production-like runnable scenario definitions.
- Use `graphs/` for graph compiler inputs.
- Use `scripts/` for maintainers' automation, validation, and release tasks.
