# Datasets

This directory contains benchmark and measurement inputs used by PhotonTrust.

## Policy

- Every dataset subtree should document provenance and redistribution status.
- Synthetic starter data may be tracked when it is necessary for examples or
  automation templates.
- Real or confidential measurement data must not be committed without explicit
  review.

## Current Areas

- `measurements/`
  - Measurement bundles and starter templates used by PIC readiness workflows.
