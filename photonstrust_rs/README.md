# photonstrust_rs

This is an experimental Rust acceleration component for selected PhotonTrust
paths.

## Current Role

- optional acceleration for event processing and yield estimation paths
- not required for the default Python-first product workflows

## Repository Hygiene Notes

- `target/` is generated and ignored
- local build logs should not be tracked
- this component should remain clearly documented as optional until it is part of
  the supported public installation path
