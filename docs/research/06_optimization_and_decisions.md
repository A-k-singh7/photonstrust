# Optimization and Decision Outputs

This document defines the optimization layer and decision artifacts.

## Objectives
- Maximize key rate and fidelity
- Minimize cost and resource usage
- Respect constraints (detectors, memories, latency)

## Output Types
- Optimal repeater spacing
- Purification schedule
- Sensitivity map: "most impactful parameter"

## Methods
- Grid search for spacing
- Multi-objective scoring with weights
- Sensitivity via finite differences

## Quality Checklist
- Deterministic results with seeds
- Clear recommendation summary
- Confidence intervals in decision outputs
