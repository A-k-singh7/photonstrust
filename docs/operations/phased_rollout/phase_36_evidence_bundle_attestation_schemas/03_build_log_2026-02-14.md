# Phase 36 - Evidence Bundle Attestation + Schema Contracts - Build Log

Date: 2026-02-14

## Summary
Added strict JSON schema contracts and validation gates for:
- workflow chain reports (`pic.workflow.invdesign_chain`)
- evidence bundle manifests (`photonstrust.evidence_bundle`)

This hardens the platform’s evidence outputs against silent contract drift.

## Changes (By File)

### Schemas
- `photonstrust/schemas/photonstrust.pic_workflow_invdesign_chain_report.v0.schema.json`
  - Workflow chain report contract (v0.1).
- `photonstrust/schemas/photonstrust.evidence_bundle_manifest.v0.schema.json`
  - Evidence bundle manifest contract (v0.1).

### Engine/Library
- `photonstrust/photonstrust/workflow/schema.py`
  - Schema path helpers for workflow and bundle artifacts.
- `photonstrust/photonstrust/workflow/__init__.py`

### Tests
- `photonstrust/tests/test_workflow_chain_report_schema.py`
  - Runs the workflow chain via API and validates `report` against schema.
- `photonstrust/tests/test_evidence_bundle_manifest_schema.py`
  - Downloads bundle zip and validates embedded `bundle_manifest.json` against schema.

## Notes
- `artifact_relpaths` maps can contain `null` values (optional artifacts), so the workflow schema allows `string|null` values for relpaths.
