# Phase 29 - Build Log (2026-02-14)

## Summary
Integrated PIC layout + LVS-lite + SPICE export seams into the managed API + web UI workflow surface.

## Backend (API)
Updated:
- `photonstrust/api/server.py`

Added endpoints (all run-registry integrated):
- `POST /v0/pic/layout/build`
  - writes: `layout_build_report.json`, `ports.json`, `routes.json`, `layout_provenance.json`, optional `layout.gds`
  - run_type: `pic_layout_build`
- `POST /v0/pic/layout/lvs_lite`
  - supports `layout_run_id` to reuse sidecars without re-posting large JSON
  - writes: `lvs_lite_report.json`
  - run_type: `pic_lvs_lite`
- `POST /v0/pic/spice/export`
  - writes: `spice_export_report.json`, `netlist.sp`, `spice_map.json`, `spice_provenance.json`
  - run_type: `pic_spice_export`

Also improved artifact serving:
- `.sp/.cir/.log/.txt/.md` served as `text/plain; charset=utf-8`.
