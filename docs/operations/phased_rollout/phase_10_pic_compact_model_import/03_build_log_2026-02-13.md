# Build Log

## Metadata
- Work item ID: PT-PHASE-10
- Date: 2026-02-13

## Summary
Added a conservative Touchstone (S2P) ingestion path for importing 2-port compact
models into ChipVerify PIC netlists, plus a wavelength sweep runner and CLI
support for executing sweeps.

## Code changes

### Touchstone parsing and interpolation
- Added:
  - `photonstrust/components/pic/touchstone.py`
    - parses Touchstone header + 2-port records (S11/S21/S12/S22)
    - supports RI/MA/DB formats and HZ/KHZ/MHZ/GHZ units
    - deterministic complex interpolation with optional extrapolation

### PIC component library
- Updated:
  - `photonstrust/components/pic/library.py`
    - added `pic.touchstone_2port` component kind
    - maps forward transmission to S21 by default

### PIC sweep execution
- Updated:
  - `photonstrust/pic/simulate.py`
    - added `simulate_pic_netlist_sweep(...)`
  - `photonstrust/pic/__init__.py`
    - exported `simulate_pic_netlist_sweep`

### CLI
- Updated:
  - `photonstrust/cli.py`
    - added `--wavelength-sweep-nm` to `photonstrust pic simulate`

- The Touchstone import is deliberately conservative:
  - 2-port only
  - S-parameters only
  - reflections are not used in v1 (forward-only solver)
