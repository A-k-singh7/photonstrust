# Build Log

## Metadata
- Work item ID: PT-PHASE-17
- Title: OrbitVerify validation + diagnostics v0.1 (schema + validate endpoint + UI surfacing)
- Date: 2026-02-13

## Summary

Added a trust-focused validation surface for OrbitVerify pass envelopes:
- published a JSON Schema contract (v0.1),
- added deterministic semantic diagnostics (errors + warnings),
- exposed validation via API (`/validate`) and included diagnostics in run responses,
- surfaced diagnostics in the web UI Orbit Pass mode,
- added unit + API tests.

## Code Changes

### Schemas
- `photonstrust/schemas/photonstrust.orbit_pass_envelope.v0_1.schema.json`
  - New JSON Schema for config-first Orbit pass envelope payloads.

### Engine (Orbit)
- `photonstrust/photonstrust/orbit/schema.py`
  - `validate_orbit_pass_config(...)` mirrors graph schema behavior (`jsonschema` optional at runtime).
- `photonstrust/photonstrust/orbit/diagnostics.py`
  - `validate_orbit_pass_semantics(...)` returns structured diagnostics compatible with existing graph diagnostics formatting.

### API
- `photonstrust/photonstrust/api/server.py`
  - Added `POST /v0/orbit/pass/validate`.
  - Updated `POST /v0/orbit/pass/run` to:
    - validate config against schema (when available),
    - compute semantic diagnostics,
    - include `diagnostics` in successful responses.

### Web UI
- `photonstrust/web/src/photontrust/api.js`
  - Added `apiValidateOrbitPass(...)`.
  - Extended `apiRunOrbitPass(...)` to accept `requireSchema`.
- `photonstrust/web/src/App.jsx`
  - Orbit Pass mode now includes:
    - topbar `Validate` action,
    - right-side `Validate` tab,
    - diagnostics rendering for validate + run outputs,
    - `Require JSON Schema on validate/run` toggle.

### Tests
- `photonstrust/tests/test_orbit_diagnostics.py`
  - Unit tests for semantic diagnostics.
- `photonstrust/tests/test_api_server_optional.py`
  - Added validate endpoint smoke test.
  - Assert run endpoint includes diagnostics.

## Notes

- This phase improves trust/UX without changing OrbitVerify’s config-first model.
- No file serving, auth, or orbit propagation was added.
