# Phase 28 - Build Log (2026-02-14)

## Summary
Added deterministic SPICE export for PIC graphs plus an optional ngspice batch runner seam. This extends PhotonTrust's adoption surface into existing EDA workflows while keeping the open-core tool-agnostic.

## Code Changes

### New modules
- `photonstrust/spice/export.py`
  - deterministic export: `pic_circuit` graph -> `netlist.sp`
  - emits `spice_map.json` with terminal->net mapping
  - emits `spice_provenance.json` with hashes + settings
  - includes optional stub `.subckt` definitions to keep netlists self-contained
- `photonstrust/spice/ngspice_runner.py`
  - optional external tool runner seam (`ngspice` on PATH)
  - batch mode execution with `-b`, log capture (`-o`), and raw capture (`-r`)

### New schema
- `schemas/photonstrust.pic_spice_export.v0.schema.json`

### New tests
- `tests/test_pic_spice_export.py`
  - validates export report schema
  - checks deterministic netlist structure (ignoring timestamps)
- `tests/test_ngspice_runner_optional.py`
  - verifies missing-tool error OR successful execution when ngspice is present

## Notes / Limitations
- Export is a connectivity interop format. Optical physics remains handled by PhotonTrust's PIC solver and performance DRC.
- ngspice result ingestion is deferred until we standardize a `.raw`/CSV ingestion contract.
