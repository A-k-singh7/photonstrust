## Changes Implemented

### Backend (API)
- Added run registry endpoints:
  - `GET /v0/runs`
  - `GET /v0/runs/{run_id}`
  - `GET /v0/runs/{run_id}/artifact?path=<relative>`
- Enforced a server-managed runs root for API runs:
  - API run endpoints reject `output_root` overrides.
  - Configure run storage via environment variable `PHOTONTRUST_API_RUNS_ROOT` (default: `results/api_runs` under repo root).
- Run manifest writing:
  - `POST /v0/qkd/run` now writes `run_manifest.json` in the run directory and returns `manifest_path`.
  - `POST /v0/orbit/pass/run` now writes `run_manifest.json`, returns `manifest_path`, and returns `artifact_relpaths` for web linking.
- Artifact serving:
  - Uses `photonstrust/api/runs.py` safe path resolution (reject absolute paths, drive letters, and `..`, plus canonical containment checks).
  - Serves HTML/JSON/PNG/PDF with explicit media types and `cache-control: no-store`.
