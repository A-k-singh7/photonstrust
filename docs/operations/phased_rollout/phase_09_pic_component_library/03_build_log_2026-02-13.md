## Summary
Implemented a minimal PIC component library and PIC netlist simulator to support
ChipVerify "drag-drop -> compiled netlist -> executable results" workflows.

This phase adds:
- a component model library with forward matrices (2-port + coupler),
- a chain loss-budget solver,
- a general feed-forward DAG solver supporting interference,
- CLI surface for PIC simulation,
- unit tests for loss accounting and MZI interference routing,
- schema/compiler port extensions for PIC edges.
