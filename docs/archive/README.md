# Archive

This directory holds historical or point-in-time repository documents that are
still useful for provenance, audits, or change history but should not be part of
the main contributor or user navigation paths.

## What belongs here

- dated repository snapshots
- one-time capability inventories
- historical planning or audit documents that are no longer active guidance

## What should stay outside archive

- active user guides
- current contributor workflow docs
- living research references still used by current development
- operational runbooks that remain part of current release or product paths
